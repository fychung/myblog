Hugo command

Sep 13, 2022
1 minute read

1、Hugo command:

  • hugo new site XXX. 生成站点
  • cd XXX
  • git init
  • git submodule add https://github.com/.../theme_XXXX.git themes/XXXX
  • 添加XXXX到config.toml
  • hugo new posts/my-first-post.md 生成md页面
  • 编辑内容
  • hugo server -D 启动，查看

2、在线托管

  • –destination 指定生成静态文件目录，例如 ./docs
  • –buildDrafts 表示posts中被标记为drafts的post，也被包括进来生成静态文件
  • –cleanDestinationDir 静态目录中没有的文件，也不会出现在destination目录中 hugo –destination ./docs –buildDrafts –cleanDestinationDir
